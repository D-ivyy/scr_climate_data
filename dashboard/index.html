Metric Guide

How To Read These Outputs

Raw values are preserved. SCR returns small physical impact values such as 0.001238918. The dashboard defaults to a percent-style display, 0.1239%, but keeps the raw value in labels and hover text.

Percent-style display is not a confirmed vendor unit. It is a readability conversion of raw value times 100. Until SCR confirms the exact product-facing label, treat it as a display view, not a final financial percentage.

Hazards have two signals. Quantified value impact is one signal. Indicator severity ratings are another. If a hazard says "not quantified," SCR did not return a numeric hazard value impact for that hazard in the selected scenario/horizon.

Missing magnitude is not missing meaning. Some indicator rows have a rating but no numeric value or unit. Those rows are still useful for severity screening, but the dashboard marks the magnitude as "rating only."
